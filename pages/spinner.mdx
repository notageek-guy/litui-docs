import { Spinner } from "@manishxdev/litui";

The `Spinner` component from `@manishxdev/litui` package provides a loading spinner animation for indicating that content is being loaded in web applications.

## Import

To use the `Spinner` component, first, import it from the `@manishxdev/litui` package:

```jsx
import { Spinner } from "@manishxdev/litui";
```

## Examples

<Spinner color="red" size={20} />
<Spinner color="blue" size={30} />
<Spinner color="pink" size={40} />
<Spinner color="green" size={50} />

## Props 

It supports props like color, size
