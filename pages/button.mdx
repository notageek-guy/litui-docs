import { Button } from "@manishxdev/litui";

The `Button` component from `@manishxdev/litui` package provides a customizable button element for use in web applications.

## Use

To use the `Button` component, first, import the button from `@manishxdev/litui` package:

```js copy
import { Button } from "@manishxdev/litui";
```

## Variants

Primary

<Button variant="primary">Click Me </Button>
```js copy 
<Button variant="primary">Click Me </Button>
```

Secondary

<Button variant="secondary">Click Me </Button>

```js copy
<Button variant="secondary">Click Me </Button>
```

Danger

<Button variant="danger">Click Me </Button>

```js copy 
<Button variant="danger">Click Me </Button>
```

Success

<Button variant='success'>Click Me </Button>

```js copy 
<Button variant='success'>Click Me </Button>
```
