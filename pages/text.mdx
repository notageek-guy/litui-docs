The `Text` component is a flexible container for displaying text content. It allows you to customize typography, color, and spacing using styled-components and styled-system.

## Installation

To install the `Text` component, you can use a package manager like npm or yarn:

````bash
import {Text} from '@manishxdev/litui'
# or


# Custom Text Component Example

To demonstrate the usage of the `Text` component from Lit UI, you can create a custom text component with specific styles.

```jsx
import React from "react";
import { Text } from '@manishxdev/litui';

const MyComponent = () => {
  return (
    <div>
      <h1>Custom Text Component Example</h1>
      <Text fontSize="24px" color="blue" mt={2}>
        This is a customized text component using Lit UI.
      </Text>
    </div>
  );
};

export default MyComponent;
````
