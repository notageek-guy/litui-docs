import {Avatar } from '@manishxdev/litui'





The `Avatar` component from `@manishxdev/litui` package displays a user's profile picture or icon in web applications.

## Import

To use the `Avatar` component, first, import the Avatar component from the `@manishxdev/litui` package:

```js copy
import {Avatar} from '@manishxdev/litui'
```


## Example


```js copy 
<Avatar src='https://api.dicebear.com/7.x/adventurer/svg?seed=Angel' alt='test'  />
```
