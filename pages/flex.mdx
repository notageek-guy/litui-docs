import { Flex, Box } from "@manishxdev/litui";

The `Flex` component provides a flexible container to distribute space and align items in web applications. It is based on the `styled-components` library and utilizes the `styled-system` package for flexbox-related styles.

## Installation

To use the `Flex` component, first, import the `Flex` component from `@manishxdev/litui` package:

```js copy
import { Flex } from "@manishxdev/litui";
```

## Example

<Flex >
<Box bg='red' width={"50px"} height={"40px"}/>
<Box bg='blue' width={"50px"} height={"40px"}/>
<Box bg='pink' width={"50px"} height={"40px"}/>

</Flex>
