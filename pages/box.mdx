import {Box} from '@manishxdev/litui'

The `Box` component from `@manishxdev/litui` package provides a versatile layout container for displaying content in web applications.

## Import

To use the `Box` component, first, import it from the `@manishxdev/litui` package:

```jsx
import { Box } from '@manishxdev/litui';
```

## Examples

<Box bg='red' width={"50px"} height={"40px"}/>
<Box bg='blue' width={"50px"} height={"40px"}/>
<Box bg='pink' width={"50px"} height={"40px"}/>
