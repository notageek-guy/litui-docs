import {Stack,Box} from '@manishxdev/litui';

`HStack` is a layout component that arranges its children in a horizontal row or vertical. It is implemented using styled-components and styled-system, allowing for easy styling and customization.

## Import 
```js copy 
import {Stack} from '@manishxdev/litui';
```

## Examples 

```jsx

const Example = () => {
    return <Stack>
    <Box>1</Box>
    <Box>2</Box>
    <Box>3</Box>
    </Stack>
}
```
