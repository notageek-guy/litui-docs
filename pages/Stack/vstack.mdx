import { VStack, Box } from "@manishxdev/litui";

`HStack` is a layout component that arranges its children in a vertical direction. It is implemented using styled-components and styled-system, allowing for easy styling and customization.

## Import

```js copy
import { VStack } from "@manishxdev/litui";
```

## Examples

```jsx
const Example = () => {
  return (
    <VStack>
      <Box>1</Box>
      <Box>2</Box>
      <Box>3</Box>
    </VStack>
  );
};
```
