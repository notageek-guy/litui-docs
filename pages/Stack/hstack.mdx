import {HStack,Box} from '@manishxdev/litui';

`HStack` is a layout component that arranges its children in a horizontal row. It is implemented using styled-components and styled-system, allowing for easy styling and customization.

## Import 
```js copy 
import {HStack} from '@manishxdev/litui';
```

## Examples 

```jsx

const Example = () => {
    return <HStack>
    <Box>1</Box>
    <Box>2</Box>
    <Box>3</Box>
    </HStack>
}
```
